# Trivia Game

A simple command-line trivia game built with JavaScript.

## What it does

This is a quiz game that runs in your terminal. You answer questions, get a score, and race against the clock.

## How to run it

1. Make sure you have Node.js installed on your computer
2. Open your terminal
3. Go to the project folder
4. Run this command:

```
node trivia.js
```

## How to play

- Read each question carefully
- Type A, B, C, or D for your answer
- Press Enter
- You have 15 seconds per question
- Try to get the highest score!

## What's inside

The game has 5 questions about general knowledge. Each question has 4 options to choose from.

## Features

- Timer for each question
- Shows if your answer is right or wrong
- Keeps track of your score
- Lets you play again when you finish

## Requirements

- Node.js (version 12 or higher)
